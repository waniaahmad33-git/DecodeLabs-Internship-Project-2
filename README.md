# DecodeLabs-Internship-Project-2
Project 2 of DecodeLabs Artificial Intelligence Internship. A machine learning classification model built using supervised learning techniques.
🤖 Data Classification Using AI

This project was completed as part of the DecodeLabs Artificial Intelligence Internship Program.

The purpose of this project is to understand the basics of supervised learning by building a simple classification model. The model is trained on a dataset and then used to make predictions on new data.

🎯 Project Goals

• Load a dataset

• Split data into training and testing sets

• Train a classification model

• Test the model's performance

• Make predictions

🛠️ Tools Used

• Python

• Pandas

• Scikit Learn

📂 Files Included

• classification.py , Main project code

• dataset.csv , Dataset used for training

• report.pdf , Project report

• screenshots , Project output screenshots

▶️ How to Run
Install required libraries
pip install pandas scikit-learn
Run the Python file
python classification.py
📸 Project Output

Screenshots of code execution and results are available in the screenshots folder.
